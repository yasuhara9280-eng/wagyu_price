# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
- リモートリポジトリとはネット上にあるファイルやディレクトリを保管しておく場所
- PC上（デスクトップ等）にあるファイルやディレクトリを保管しておく場所

## プッシュとマージの違いは何でしょうか？
- ローカルリポジトリの内容をリモートリポジトリにアップする事（プッシュ）
- 別のブランチを今いるブランチに取り込む事（マージ）



## コミットとプッシュの違い
- コミットとはファイルの変更内容をメッセージと共に保存すること（ゲームでいうセーブポイント）
- プッシュとはローカルリポジトリの内容をリモートリポジトリにアップすること


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
- 相手にも自分にもわかりやすく自分が行ったことを簡潔に書く



## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- 自分で直接mainにマージをするのか、誰かのチェックを通してマージをするのか



## コンフリクトを起こしてしまった場合、どう対処すべきですか？
- 先にマージした変更内容を取り込む
- 後にマージした変更内容を取り込む
- どちらの変更内容も取り込む
