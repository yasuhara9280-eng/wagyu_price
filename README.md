# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
・リモートリポジトリとはネット上にあるファイルやディレクトリ
・PC上（デスクトップ等）にあるファイルやディレクトリ

## プッシュとマージの違いは何でしょうか？
・ローカルリポジトリの内容をリモートリポジトリにアップすること（プッシュ）か別のブランチをmainに取り込む（マージ）か。

## コミットとプッシュの違い
・途中でのセーブ（コミット）かローカルからリモートにアップするか（プッシュ）

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
・相手にも自分にもわかりやすく自分が行ったことを簡潔に書く

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
・自分で直接mainにマージをするのか、誰かのチェックを通してマージをするのか

## コンフリクトを起こしてしまった場合、どう対処すべきですか？
・先にマージをした変更内容を取り込む
・後からマージした変更内容を取り込む
・どちらの変更内容も取り込む
